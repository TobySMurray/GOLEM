Downloadable and playable at: https://tobymurray.itch.io/golem-alpha

Golem is a top-down bullet-hell roguelike, featuring a unique character-swapping mechanic where the player takes control of enemies, while managing their resources. 
Play as each golem, with their own skills and item upgrades.

This game was originally created for the Ludum Dare 48 gamejam in 72 hours. It is now being expanded and polished beyond the scope of the jam.

Developers: Andrew Cunningham, Toby Murray, Lee Zeitz

Music: Keaton House

Art: Penusbmic, Liam Cunningham, Kevin House

Golem is built with the Godot Game Engine.
